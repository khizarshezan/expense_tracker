# Smart Expense Tracker

## Overview
A full-stack web application for tracking personal income and expenses with real-time charts and category analysis.

## Features
- Add income and expense transactions
- Real-time balance calculation
- Spending breakdown by category (donut chart)
- Monthly income vs expense comparison (bar chart)
- Delete transactions
- Data stored in MySQL database

## Tech Stack
- **Python** — Backend logic
- **Flask** — Web framework
- **MySQL** — Data storage
- **Chart.js** — Interactive charts
- **Bootstrap** — Frontend styling
